30 /*!
31 * \brief Event Detection by comparison of signal to RMS Noise level.
32 * event detection: fill a circular buffer with complex samples and
33 * search for peaks nsigma above the RMS of the data stream
34 * input:
35 * complex vector of I/Q samples
36 * parameters
37 * 1. Vector length
38 * 2. Number of sigma to declare an event
39 * 3. Bandwidth used to unwind the time of the event in circular buffer
40 * 4. Estimated time it takes for sample to go from input of horn to block
41 * 5. Mode: 1: Monitor, just pass input data,
42 * 2: Detect events and repeatedly output the last event
43 * output:
44 * 1: Vector of complex I/Q samples
45 * Event is tagged with three floating point values:
46 * 1. Modified Julian Date of Event
47 * 2. Peak intensity
48 * 3. RMS of data stream near event
49 * \ingroup radio_astro
50 *
51 */
